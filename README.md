# Wishlist API Service

Простой бэкенд для создания списков желаний и их бронирования. 
Написано на Go + Postgres.

## Стек
- Go 1.21 (chi, sqlx, jwt-go, bcrypt)
- PostgreSQL 15
- Docker Compose

## Как запустить
Все максимально просто:
1. Клонируешь репо.
2. Делаешь `cp .env.example .env`.
3. Запускаешь `docker-compose up --build`.

База инициализируется автоматически при первом запуске приложения. Приложение будет доступно на `:8080`.

## Особенности реализации
- **Атомарное бронирование**: Используется прямой SQL UPDATE с проверкой состояния. Это исключает двойное бронирование (race condition) без лишних блокировок.
- **Безопасность**: Пароли в bcrypt, доступ к CRUD операциям через JWT.
- **Публичный доступ**: Списки доступны по уникальным коротким токенам (не ID), что предотвращает перебор.
- **Graceful Shutdown**: Приложение корректно завершает соединения при остановке контейнера.

## API Эндпоинты
- `POST /auth/register` — регистрация.
- `POST /auth/login` — получение токена.
- `GET /share/{token}` — публичный просмотр вишлиста.
- `POST /share/{token}/book/{itemID}` — анонимное бронирование подарка.
- `/api/wishlists` — (Private) набор методов для управления своими списками.

## Фронтенд
Зайдите на `http://localhost:8080/index.html?t=ВАШ_ТОКЕН` после создания вишлиста, чтобы увидеть публичную страницу.
